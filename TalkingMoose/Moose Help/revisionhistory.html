Revision History

3.6 Thank you for explaining
  • Ignore screen saver more often.
  • Some touch-ups and improvements to animations, support for larger animations in more efficient way.
  • Cleaned up the user interface, modernized it, and in the process removed a few options.
  • Added a fallback mode that detects when one of the lip-sync-unaware modern voices is used and just generate random mouth positions.
3.5.7 Beware the Weeping Angels

  • Workarounds for more Leopard speech synthesis bugs, mainly mine.

  • Made the crash reporter report crashes on Leopard and improved the feedback window.
  • Fixed detection of the screen saver, which Apple completely changed in Leopard.
